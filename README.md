# Student-dropout-analysis
## Project Overview
This project analyzes factors influencing student dropout risk using student demographic, academic performance, attendence, and socio-economic data. Excel was used for data cleaning, while power BI was used for analysis and dashboard visualization. 
## Data Columns
Student number
students name
Age, Age group, Gender
Overall performance
Overall attendance
Attendence status (Good / Bad)
Family income per year
Income class
Distance to school (km)
Distance category  (Near / Far)
Dropout risk
## Key KPIs
Total number of students 
Average attendance rate 
Average academic performance
Percentage of high-risk students
Dropout risk by income class
Dropout risk by attendance status
Dropout risk by distance to school
Dropout risk by age group
Dropout risk by gender
## Tools Uesd 
Microsoft Excel ( Data Cleaning )
Power BI  (Dashboard & visualization )
## Deliverables
Cleaned Excel dataset
Power BI dashboard file
Dashboard screenshot
## Outcome
The dashboard highlights key factors contributing to students dropout risk and helps identify high-risk student groups for early intervention.
